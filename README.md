# TO DO App :clipboard:

TODO App is a task management app to help you stay organized and manage your day-to-day.

## Installation

Clone the repo to install the app

```bash
git clone https://github.com/liaz98/todoApp.git
```
To run the command
```bash
python manage.py runserver
```
